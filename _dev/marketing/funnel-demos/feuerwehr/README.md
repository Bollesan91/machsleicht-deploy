# Funnel-Demo Feuerwehr

Annotiertes Doppel-Mockup: **Einladungs-Bild + Partyseite mit Wunschliste** für das Motto Feuerwehr. Zeigt den vollen Funnel in einer Grafik — Planung → Einladung → Partyseite → Wunschliste (Affiliate).

## Was ist das?

Side-Effekt einer Konversation über die Einladungs-Frage (27.05.). Ursprünglich didaktisches Mockup, um zu zeigen, dass „Einladung" und „Partyseite" nicht zwei Tools sind, sondern eine Sache: das Bild ist ein Render der Partyseiten-Hero, der eine Link führt zur Partyseite, die Wunschliste ist der einzige echte Klick-Anlass (weil man aus einem Screenshot kein Geschenk kauft).

## Möglicher Use-Case (nicht entschieden)

- Pinterest-Story-Pin (DACH-Eltern, kostenloser Akquisekanal)

## Status

**Design-auf-Papier, nicht Bau-jetzt.** Bei 0 Planer-Traffic öffnet die Partyseite niemand — das Asset hilft erst, wenn der Planer zieht. Funnel-Axiom bleibt: Hero = „Kindergeburtstag planen", Einladung sekundär.
